# Raycasting Engine (C++ & SDL2)

Проект псевдо-3D движка на основе алгоритма Raycasting, транскрибированный с Python (Coder Space) на C++ с использованием библиотеки SDL2.

### Особенности
- Написано на чистом C++ без использования тяжелых IDE (только Notepad++)
- Собственные скрипты сборки через батники
- Высокая производительность за счёт низкоуровневого языка
- Эффект тумана (затемнение по глубине)

### Как запустить
1. Убедитесь, что у вас установлен компилятор (MinGW) и библиотека SDL2 по пути `C:/SDL/` (сверьтесь с батниками и при необходимости перепишите)
2. Запустите `debug.bat` для быстрой компиляции и запуска в режиме отладки.
3. Запустите `compile.bat` для сборки полноценного продукта.

### Управление
- **W, S, A, D** — передвижение.
- **Q, E** — поворот камеры.
